Setup (Non-VSCode)#

  • Follow Rojo installation instructions - use Rojo 7

  • Open codebase in your preferred text editor

  • Open console in root directory of the repository

  • Run the rojo serve command

  • In Roblox Studio, open Rojo and click the Connect button

VSCode Setup#

  • Install Rojo

  • Select MANAGE IT FOR ME on the options to install the plugin

  • Open the project in VSCode

  • Build latest place files by running buildplaces.bat

  • Click Start Rojo near the bottom and select the project file for which place you want to work in.

  • In Roblox Studio, open Rojo and click the Connect button

Recommended Plugins#

  • Tag Editor for use with the component system and CollectionService

  • Roblox LSP if using VSCode

  • TabNine if using VSCode

  • TestEZ Companion if using VSCode
