Load

Purpose of this is primarily to use in components. We want x async steps to execute before the component is usable i.e. has loaded, so allow AwaitLoaded to block any coroutines until it's loaded OR it times out. Timeouts prevent memory leaks due to dead coroutines.

In short, makes awaiting object loading much easier.
