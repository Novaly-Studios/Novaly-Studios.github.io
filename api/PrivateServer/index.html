GetReservedServerIDs#

</>
PrivateServer:GetReservedServerIDs() â†’Â ()

This will attempt to get saved reserved server codes from the DataStore.

  • If there is no result, it will attempt to get and save new ones.
  • If there is an error, it will return nothing.
  • If successful, it will return a map of place IDs to reserved server codes for this private server.
